本示例使用到的资源：
TIM3 CH1     --   PB4
TIM3 CH2     --   PB5
TIM5 CH1     --   PB8

本示例的功能流程：
1、上电配置时钟（PLL 48MHz）；
2、TIM3的配置：
   TIM3从模式选择为门控模式，触发源为TI2FP2，上升沿有效；
   TIM3主模式选择为使能模式；
   
   TIM3通道1比较输出模式为翻转模式；
   
3、TIM5的配置：
   TIM5从模式选择为门控模式，触发源为ITR1；
   
   TIM5通道1比较输出模式为翻转模式；
 
4、当TIM3通道2上检测到有效的上升沿后，触发TIM3、TIM5通道1上同步输出占空比为50%的波形；

注意：
1、因TIM3从模式触发源为上升沿有效，故通道2事先应保持低电平；
2、各TIM输出波形如下：
