本示例使用的资源：
ADC_IN2      --   PA3
VREFBUF      --   PA0

本示例的功能实现流程：
通过ADC中断的方式获取ADC通道2的输入电压值，通过内部VREFBUF参考电压计算通道2的转换电压。

1、上电配置系统时钟为RCH 16MHz；
2、初始化PA3，ADC采集此引脚电压；初始化PA0模拟模式，作为VREFBUF输出；
3、初始化VREFBUF；
4、初始化ADC：
     1）同步时钟源，不分频，采样时间119adc_clk；
     2）正向扫描； 
     3）转换通道选择通道2； 
     4）软件触发，单次扫描转换模式；
     5）参考电压选择VERFBUF;
5、ADC校准;
6、启动ADC转换，中断模式;
7、等待转换完成，读取ADC转换数据;
8、重复上述步骤，读取通道ADC转换数据，读取10次并求和取平均;
9、外部通道采样平均值转换成电压值;
