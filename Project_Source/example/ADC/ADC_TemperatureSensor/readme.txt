本示例使用的资源：
VREFBUF     --  PA0
USART1 TX   --  PA2
USART1 RX   --  PA1

本示例的功能实现流程：
通过查询方式获取VTS通道的采集值，通过内部VREFBUF参考电压计算温度值，通过串口打印出来。

1、上电配置系统时钟为RCH 16MHz；
2、初始化PA0模拟模式，作为VREFBUF输出；
3、初始化VREFBUF；
4、初始化USART1，配置波特率9600；
5、初始化ADC：
     1）同步时钟源，不分频，采样时间119adc_clk；
     2）正向扫描；
     3）转换通道选择通道12(VTS)； 
     4）软件触发，单次扫描转换模式；
     5）参考电压选择VERFBUF;
6、ADC校准；
7、启动ADC转换，查询模式；
8、等待转换完成，读取ADC转换数据;
9、重复上述步骤，读取通道ADC转换数据，读取10次并求和取平均;;
10、并将VTS采样平均值转换成温度值，通过串口打印输出;
