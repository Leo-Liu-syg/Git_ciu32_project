本示例使用的资源：
LED1            --   PB14
ADC_IN11        --   PB11
VREFBUF         --   PA0

本示例的功能实现流程：
模拟看门狗监控ADC通道11的输入电压，当输入电压超出阈值范围，LED1灯闪烁，未超出阈值LED1常灭。

1、上电配置系统时钟为RCH 16MHz；
2、初始化PB11，ADC采集此引脚电压；初始化PA0模拟模式，作为VREFBUF输出；
3、初始化LED1常灭；
4、初始化VREFBUF；
5、初始化ADC：
     1）同步时钟源，不分频，采样时间119adc_clk；
     2）正向扫描；
     3）转换通道选择通道11；
     4）单次扫描转换模式，软件触发；
     5）参考电压选择VERFBUF;
     6）选择模拟看门狗1，监控通道11，设置高、低阈值；
6、ADC校准；
7、启动ADC转换；
8、判断ADC转换是否超出阈值，
     1）超出阈值，LED1闪烁；
     2）未超出阈值，LED1常灭；
