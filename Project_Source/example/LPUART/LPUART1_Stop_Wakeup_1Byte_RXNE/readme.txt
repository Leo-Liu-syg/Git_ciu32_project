本示例使用的硬件资源：
LPUART1 TX      --   PB11  
LPUART1 RX      --   PB10

LPUART通信参数配置:
  -- 波特率:   9600
  -- 字符长度：8bits
  -- 停止位：  1bits
  -- 奇偶校验：无
  -- 硬件流控：禁止


本示例用于演示LPUART1 从Stop模式唤醒的过程，唤醒源为接收一字节数据
1、系统时钟使用RCH的2分频：8MHz
2、配置LPUART1的GPIO ；
3、初始化LPUART1，时钟源为RCH ；
4、使能接收一帧数据中断，使能LPUART1唤醒，完成配置后，等待3s 后进入Stop 模式 ；
5、接收到一帧数据时，从Stop模式唤醒，唤醒后进入中断 ；
6、使用中断接收的方式，在接收中断中将读取接收数据；
7、接收中断处理完成后，发送用于唤醒的接收数据。
