本示例使用的硬件资源：
USART1 TX      --   PA11  
UART2  TX      --   PD3

单线半双工模式下，默认使用TX引脚收发。
PA11连接PD3

USART通信参数配置:
    -- 波特率:   115200
    -- 字符长度：8bits
    -- 停止位：  1bit
    -- 奇偶校验：无
    -- 硬件流控：禁止


本示例的功能流程：
1、上电配置系统时钟为PLL（48MHz）；
2、初始化USART使用的GPIO，使用复用开漏模式，使能上拉电阻；
3、初始化USART1、UART2，使能单线半双工模式；
4、执行USART1发送8个数据UART2接收8个数据；
5、执行UART2发送8个数据USART1接收8个数据。
