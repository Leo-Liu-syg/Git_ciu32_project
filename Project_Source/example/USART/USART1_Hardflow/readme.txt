本示例使用的硬件资源：
USART1 TX         --   PA11  
USART1 RX         --   PA12
USART1 RTS        --   PA10  
USART1 CTS        --   PA9


USART1 通信参数配置:
  -- 波特率:     115200
  -- 字符长度：  8bits
  -- 停止位：    1bit
  -- 奇偶校验：  无
  -- 硬件流控：  使能


本示例的功能流程：
1、上电配置系统时钟为PLL（48MHz）；
2、初始化USART1使用的GPIO；
3、初始化USART1,使能硬件流控；
4、使用查询方式发送数据；
5、查询到接收数据非空时，读取接收数据，并发送此接收数据。
