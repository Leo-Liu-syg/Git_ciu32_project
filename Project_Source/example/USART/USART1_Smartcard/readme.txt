本示例使用的硬件资源：
USART TX        --  PA11   -- 智能卡 I/O
USART CK        --  PA10   -- 智能卡 CK
GPIO            --  PB10   -- 智能卡 RST
GPIO            --  PB11   -- 插卡检测引脚
GPIO            --  PD2   -- 电源控制

本示例实现了冷启动以及命令交互流程，适用于T=0：
1、上电配置系统时钟为PLL（48MHz）
2、初始化GPIO，初始化USART为智能卡主模式
3、等待插卡检测
4、检测到卡片插入后，进入冷复位流程，过程中接收Answer-to-Reset信息并保存
5、与卡片进行一次交互
6、交互过程中使用接收超时功能监测数据流
7、交互失败时执行去激活流程
