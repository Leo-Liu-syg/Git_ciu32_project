本示例使用的资源：
ADC_IN2      --   PA3
VREFBUF      --   PA0

本示例的功能实现流程：
使用TIM3的TRIG_OUT作为硬件触发源，通过DMA中断的方式获取ADC通道2的输入电压值，通过VREFBUF内部参考电压计算通道2的转换电压。

1、上电配置系统时钟为RCH 16MHz；
2、初始化PA3，ADC采集此引脚电压；初始化PA0模拟模式，作为VREFBUF输出；
3、初始化VREFBUF；
4、初始化TIM3；
5、初始化和配置DMA；
6、初始化ADC：
   1）同步时钟源，不分频，采样时间119adc_clk；
   2）正向扫描；
   3）转换通道选择通道2；
   4）上升沿触发，触发源为TIM3 TRIG_OUT；
   5）参考电压选择VERFBUF;
7、ADC校准；
8、启动ADC转换，DMA传输模式(软件触发)
9、DMA中断检查通道是否转换完成；
10、获取DMA传输的ADC转换数据，并计算电压值。
