本示例使用到的资源：
TIM3 CH1     --   PB4
TIM3 ETR     --   PC10

本示例的功能流程：
1、上电配置时钟（PLL 48MHz）；
2、配置TIM3通道1输出单脉冲；
   ETR为单脉冲输出的触发源，上升沿有效;
3、当检测到触发信号有效沿时，输出单脉冲波形；

注意：
1、通道2触发信号的有效沿到通道1输出单脉冲信号之间的时间间隔为：Delay；
   通道1输出单脉冲信号的宽度为：Pulse；
   
   TIM3的计数时钟 = 1MHz，故分频比配置值 psc_value = (((SystemCoreClock) / 1000000) - 1);
   
   Delay =  CC1/TIM3计数时钟 = (16383 / 1000000) 秒
   Pulse = (ARR - CC1)/TIM3计数时钟 = ((65535 - 16383) / 1000000) 秒
