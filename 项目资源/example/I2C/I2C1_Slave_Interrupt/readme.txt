本示例使用到的资源：
LED1         ------ PB14
I2C1_SCL     ------ PB8
I2C1_SDA     ------ PB9


本示例的功能流程：
1、上电配置系统时钟（PLL作为时钟源）为48M；
2、初始化LED1的GPIO，并初始状态为常灭，初始化I2C1的GPIO；
3、设置I2C1的工作时钟为PLL（48M），初始化I2C1为从机模式，并开启从模式时钟延长功能；
4、初始化NVIC，使能I2C1的中断并配置中断优先级；
5、使能I2C1的地址匹配中断、NACK检测中断及停止位检测中断；
6、控制LED1常亮；
7、若进入中断服务程序：
   - 地址匹配时，根据读写位方向分别使能发送中断或接收中断；
   - TXIS标志置位时，发送数据；
   - RXNE标志置位时，接收数据；
   - NACKF标志置位时，清除NACKF标志；
   - STOPF标志置位时，清除STOPF标志。

注意：
该程序与I2C1_Master_Poll配套使用，当LED1点亮时，再控制主机发起通信。
